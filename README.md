# Django- Todo App
This Project is a basic TODO-WEBAPP create with python django framework.Where user can manage their daily task (Routine or Track progress) .

Features:-
> Create, update and delete tasks
> Mark tasks as complete/incomplete
> updates tasks anytime

For frontend i have used HTML CSS and Bootstrap .(Bootstrap with help of internet not very familiar with Bootstrap still in learning phase.)


Working logic :-
1. User Submits form
2. Django reconize POST request
3. Data is stored in DB
4. UI with CSS HTML And Bootstrap .


How can new learner can use this project .
1. git clone https://github.com/abyshk-np/todo-app.git ( run this command in terminal)
2. cd Todo (Go to project folder)
3. python -m venv env ( Create virtual environment)
4. env\Scripts\activate (env\Scripts\activate )
5. pip install django (Install Django)
6. python manage.py migrate ( Run migration )
7. python manage.py runserver (You can run it now)
